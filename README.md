🌍 Triplio — Premium Global Expedition Framework

> **Uncover the World's Natural Wonders.**  
Triplio is a modern, responsive, and visually immersive travel landing page designed for luxury expedition brands and nature enthusiasts. Built with clean HTML5 semantic structure, modern CSS styling, and interactive UI components.

---

## ✨ Features

* **🎥 Video Hero Background:** Features an autoplaying, looping background video with dark overlay for enhanced contrast and readability.
* **📅 Interactive Booking Widget:** Allows users to select destination location, travel dates, and guest counts right from the hero section.
* **📊 Animated Stats Bar:** Includes dedicated data attributes (`data-target`) ready for JavaScript counter animations.
* **🌲 Ecosystem Catalog & Filtering:** Interactive destination gallery with filter tabs (*Alpine Peaks, Tropical Wilds, Arid Canyons*).
* **📱 Fully Responsive Navbar:** Hamburger menu toggle ready for mobile navigation drawers and seamless multi-device browsing.
* **🎨 Modern Typography & Icons:** Integrated with [Font Awesome 6](https://fontawesome.com/) for rich icon typography.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic structure for optimal accessibility and SEO.
* **CSS3:** Custom styles (`style.css`) for responsive layouts, hero parallax, and dynamic controls.
* **Font Awesome 6.4.0:** Icon package for navigation, booking attributes, and social media links.
* **JavaScript (Optional/Planned):** Powers statistics counter animations, ecosystem filtering, and mobile menu toggles.
